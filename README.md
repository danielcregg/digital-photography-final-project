A digital photography portfolio showcasing work across multiple photographic disciplines, submitted as a final project. The collection demonstrates skills in long exposure, sequence narrative photography, and competition-level composition.

## Overview

This repository contains a curated selection of photographs created for a digital photography course final project. Each image represents a different photographic technique and artistic approach, accompanied by written descriptions providing context and creative rationale.

## Features

- **Long Exposure Photography** — Night photography using long exposure techniques to capture motion and light trails
- **Sequence Photography Narrative** — A visual narrative told through sequence photography, documenting a process or story
- **Competition Entry** — A photograph prepared and submitted for a photography competition
- **Written Descriptions** — Detailed descriptions and context for each photograph in an accompanying document

## Prerequisites

- Any image viewer or photo editing application (e.g., Adobe Photoshop, GIMP, IrfanView)
- A word processor capable of opening `.docx` files (e.g., Microsoft Word, LibreOffice Writer)

## Getting Started

### Installation

```bash
git clone https://github.com/danielcregg/digital-photography-final-project.git
cd digital-photography-final-project
```

### Usage

Open the individual `.jpg` files in any image viewer or photo editing application. Refer to `Photo Descriptions.docx` for detailed descriptions and context for each photograph.

```
digital-photography-final-project/
├── Long Exposure/
│   └── busyNight.jpg                    # Long exposure night photography
├── Sequence Photography Narrative/
│   └── I build a pyrimad.jpg            # Sequence narrative photograph
├── Competition Photo/
│   └── craicAgusCoel.jpg                # Competition entry photograph
└── Photo Descriptions.docx              # Written descriptions for each photograph
```
